Sizes apply per-volume. Packs are divided by estimated volume count before comparison.
How Size Limits Work
  • Format is detected from the release title extension (.cbz, .cbr, .epub, .pdf, .zip).
  • Min size — releases smaller than this are rejected as likely corrupt or incomplete.
  • Max size — releases larger than this are rejected as oversized. Set to 0 to allow any size.
  • The Unknown row applies when no extension is matched.
